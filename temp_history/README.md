# temp_history/ — Wave Archive

This folder contains snapshots of temp.md taken after each wave is completed and validated.

## Rotation Rule
After each completed wave:
1. Copy temp.md → `temp_YYYY-MM-DD_waveN.md` here
2. Clear temp.md
3. Extract validated facts → `/analysis/`
4. temp.md MUST NOT be used as reference in subsequent runs
