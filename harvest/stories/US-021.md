# US-021: Manage Gdprs (GDPR-håndtering)

**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Adresser & Data
**HTTP Verbs:** DELETE, GET, POST, PUT
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i GDPR-håndtering, oprette GDPR-håndtering, redigere GDPR-håndtering og slette GDPR-håndtering, så jeg kan håndtere og validere adressedata korrekt.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over GDPR-håndtering
- [ ] Brugeren kan søge og filtrere GDPR-håndtering
- [ ] Brugeren kan oprette ny/nye GDPR-håndtering via formular
- [ ] Brugeren kan redigere eksisterende GDPR-håndtering
- [ ] Brugeren kan slette GDPR-håndtering med bekræftelsesdialog
