**Status:** ⏸️ HOLD

**HTTP Verbs:** DELETE, GET, POST, PUT

## User Story

Som bruger vil jeg se og søge i kortvisning, oprette kortvisning, redigere kortvisning og slette kortvisning, så jeg kan styre brugeradgang og sikkerhed i systemet.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over kortvisning
- [ ] Brugeren kan søge og filtrere kortvisning
- [ ] Brugeren kan oprette ny/nye kortvisning via formular
- [ ] Brugeren kan redigere eksisterende kortvisning
- [ ] Brugeren kan slette kortvisning med bekræftelsesdialog
