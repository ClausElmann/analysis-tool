# US-016: Manage Contacts (kontakter)

**Status:** 🟡 READY
**Prioritet:** P1
**Domain:** Kunder & Tilmelding
**HTTP Verbs:** DELETE, GET, POST, PUT
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i kontakter, oprette kontakter, redigere kontakter og slette kontakter, så jeg kan administrere kundernes tilmeldingsforhold og kontaktdata.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over kontakter
- [ ] Brugeren kan søge og filtrere kontakter
- [ ] Brugeren kan oprette ny/nye kontakter via formular
- [ ] Brugeren kan redigere eksisterende kontakter
- [ ] Brugeren kan slette kontakter med bekræftelsesdialog
