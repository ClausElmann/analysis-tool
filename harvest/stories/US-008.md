**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Økonomi & Drift
**HTTP Verbs:** DELETE, GET, POST, PUT
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i salgsdata, oprette salgsdata, redigere salgsdata og slette salgsdata, så jeg kan holde styr på økonomi og driftsprocesser.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over salgsdata
- [ ] Brugeren kan søge og filtrere salgsdata
- [ ] Brugeren kan oprette ny/nye salgsdata via formular
- [ ] Brugeren kan redigere eksisterende salgsdata
- [ ] Brugeren kan slette salgsdata med bekræftelsesdialog
