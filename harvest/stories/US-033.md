**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Økonomi & Drift
**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i fildrev, oprette fildrev og slette fildrev, så jeg kan holde styr på økonomi og driftsprocesser.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over fildrev
- [ ] Brugeren kan søge og filtrere fildrev
- [ ] Brugeren kan oprette ny/nye fildrev via formular
- [ ] Brugeren kan slette fildrev med bekræftelsesdialog
