# US-004: Manage Users (brugere)

**Status:** 🟡 READY
**Prioritet:** P2
**Domain:** Brugere & Adgang
**HTTP Verbs:** DELETE, GET, PATCH, POST, PUT
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i brugere, oprette brugere, redigere brugere og slette brugere, så jeg kan styre brugeradgang og sikkerhed i systemet.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over brugere
- [ ] Brugeren kan søge og filtrere brugere
- [ ] Brugeren kan oprette ny/nye brugere via formular
- [ ] Brugeren kan redigere eksisterende brugere
- [ ] Brugeren kan slette brugere med bekræftelsesdialog
