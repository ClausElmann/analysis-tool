# US-030: Manage Imports (dataimport)

**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Adresser & Data
**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i dataimport, oprette dataimport og slette dataimport, så jeg kan håndtere og validere adressedata korrekt.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over dataimport
- [ ] Brugeren kan søge og filtrere dataimport
- [ ] Brugeren kan oprette ny/nye dataimport via formular
- [ ] Brugeren kan slette dataimport med bekræftelsesdialog
