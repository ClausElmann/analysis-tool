**Status:** ⏸️ HOLD

**HTTP Verbs:** DELETE, GET, POST, PUT
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i vejrdata, oprette vejrdata, redigere vejrdata og slette vejrdata, så jeg kan kommunikere effektivt med kunder og modtagere.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over vejrdata
- [ ] Brugeren kan søge og filtrere vejrdata
- [ ] Brugeren kan oprette ny/nye vejrdata via formular
- [ ] Brugeren kan redigere eksisterende vejrdata
- [ ] Brugeren kan slette vejrdata med bekræftelsesdialog
