# US-036: Manage Configurations (konfigurationer)

**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Brugere & Adgang
**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i konfigurationer, oprette konfigurationer og slette konfigurationer, så jeg kan styre brugeradgang og sikkerhed i systemet.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over konfigurationer
- [ ] Brugeren kan søge og filtrere konfigurationer
- [ ] Brugeren kan oprette ny/nye konfigurationer via formular
- [ ] Brugeren kan slette konfigurationer med bekræftelsesdialog
