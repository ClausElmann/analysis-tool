# US-003: Manage Profiles (profiler)

**Status:** 🟡 READY
**Prioritet:** P2
**Domain:** Brugere & Adgang
**HTTP Verbs:** DELETE, GET, POST, PUT
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i profiler, oprette profiler, redigere profiler og slette profiler, så jeg kan styre brugeradgang og sikkerhed i systemet.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over profiler
- [ ] Brugeren kan søge og filtrere profiler
- [ ] Brugeren kan oprette ny/nye profiler via formular
- [ ] Brugeren kan redigere eksisterende profiler
- [ ] Brugeren kan slette profiler med bekræftelsesdialog
