# US-042: Manage Statstidendes (statstidendeer)

**Status:** ⏸️ HOLD
**Prioritet:** P3
**Domain:** Adresser & Data
**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i statstidendeer, oprette statstidendeer og slette statstidendeer, så jeg kan håndtere og validere adressedata korrekt.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over statstidendeer
- [ ] Brugeren kan søge og filtrere statstidendeer
- [ ] Brugeren kan oprette ny/nye statstidendeer via formular
- [ ] Brugeren kan slette statstidendeer med bekræftelsesdialog
