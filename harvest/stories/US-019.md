**Status:** 🟡 READY
**Prioritet:** P2

**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i tilmeldinger, oprette tilmeldinger og slette tilmeldinger, så jeg kan administrere kundernes tilmeldingsforhold og kontaktdata.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over tilmeldinger
- [ ] Brugeren kan søge og filtrere tilmeldinger
- [ ] Brugeren kan oprette ny/nye tilmeldinger via formular
- [ ] Brugeren kan slette tilmeldinger med bekræftelsesdialog
