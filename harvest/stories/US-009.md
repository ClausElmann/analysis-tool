# US-009: Manage Prospects (salgsmuligheder)

**Status:** 🟡 READY
**Prioritet:** P2
**Domain:** Kunder & Tilmelding
**HTTP Verbs:** DELETE, GET, PATCH, POST
**Kilde:** Angular harvest → user story

---

## User Story

Som bruger vil jeg se og søge i salgsmuligheder, oprette salgsmuligheder, redigere salgsmuligheder og slette salgsmuligheder, så jeg kan administrere kundernes tilmeldingsforhold og kontaktdata.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over salgsmuligheder
- [ ] Brugeren kan søge og filtrere salgsmuligheder
- [ ] Brugeren kan oprette ny/nye salgsmuligheder via formular
- [ ] Brugeren kan redigere eksisterende salgsmuligheder
- [ ] Brugeren kan slette salgsmuligheder med bekræftelsesdialog
