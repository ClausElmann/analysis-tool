**Status:** 🟡 READY
**Prioritet:** P2

**HTTP Verbs:** DELETE, GET, POST
**Kilde:** Angular harvest → user story

## User Story

Som bruger vil jeg se og søge i poster, oprette poster og slette poster, så jeg kan kommunikere effektivt med kunder og modtagere.

---

## Acceptance Criteria

- [ ] Brugeren ser en liste over poster
- [ ] Brugeren kan søge og filtrere poster
- [ ] Brugeren kan oprette ny/nye poster via formular
- [ ] Brugeren kan slette poster med bekræftelsesdialog
